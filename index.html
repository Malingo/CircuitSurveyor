CircuitSurveyor

A Java application for visualizing the electric, magnetic, and Poynting fields near an electrical circuit.

CircuitSurveyor was created in 2010–11 by Noah Morris, a student of Oberlin College, under the advisement of Oberlin College Professor of Physics Dan Styer.

Show instructions Hide instructions to download the source code of CircuitSurveyor as a .zip file.

Right-click CircuitSurveyor.zip and choose to save the linked file to your local computer. Unzip the downloaded file into a directory. Use a text editor to edit the fifth line of 'Launch CircuitSurveyor.jnlp', by changing the 'codebase' attribute to the directory containing the file. The local copy of the program should then be launchable through Java Web Start, either by double-clicking on the .jnlp file, or by making use of the 'javaws' command-line tool.
